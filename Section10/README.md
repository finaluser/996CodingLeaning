# Postman简介
postman是一款功能强大的网页调试与发送网页Http请求的工具.能够发送各种类型的HTTP请求,附带任何数量的参数和HTTPHeader支持,支持不同的认证机制,接收到的响应语法高亮.

- postman学习中心
    - https://learning.postman.com/
